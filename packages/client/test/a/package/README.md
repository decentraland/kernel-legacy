# utils

This repository holds functions to deal with repeatedly needed tasks and logic.

## pure

Pure functions, math, promises, "atomicHelpers".

## parametrization

Global configurations of the Decentraland world that are considered "pretty static" and are unlikely to change.

## parcel

Helpers based on parametrization settings.

## network

Cached fetches and URL resolution logic.

## TaskQueue

An implementation to queue tasks and microtasks

## SafeLocalStorage

Simple wrappers for localstorage that never throw

## sleep

Returns a promise that resolves in a number of millis
